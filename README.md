# MioProxy

Icon assets for MioProxy across Windows and macOS targets.

## Contents

- `assets/icons/Win/` - Windows app, tray, and glyph SVG assets.
- `assets/icons/Mac/` - macOS app and glyph SVG assets.

## Security Notes

The current SVG assets are static vector files. They do not include scripts,
event handlers, external resource references, embedded data URIs, or secrets.
